# 💼 Eric Portfolio

Website portofolio pribadi yang menampilkan hasil karya dalam pengembangan aplikasi dan website.  
Dibuat dengan tampilan modern, minimalis, dan mudah diakses di berbagai perangkat.

## ✨ Tentang
Portofolio ini berisi proyek-proyek yang pernah saya kembangkan, mulai dari aplikasi desktop berbasis VB.NET hingga website afiliasi produk.  
Tujuan utamanya adalah menampilkan kemampuan teknis dan estetika desain yang profesional.

## 🧩 Teknologi
- HTML, CSS, JavaScript  
- Bootstrap  
- Vercel (Hosting)

## 🚀 Fitur
- Halaman portofolio interaktif
- Galeri gambar dengan overlay detail proyek
- Desain responsif & ringan
- Navigasi mudah
